# Travel Checklist

A simple, interactive web-based travel checklist to help you pack and prepare for your trips. Never forget essential items again!

## Features

- **Interactive Checklist** - Click items to mark them as packed/completed
- **Progress Tracking** - Visual progress bar shows completion percentage
- **Persistent Storage** - Your selections are saved locally in the browser
- **Sort Function** - Organize items by completed/pending status
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Multi-language** - Available in Estonian and English

## Categories

The checklist is organized into practical categories:

- **Clothing** - Essential wear items
- **Footwear** - Shoes and boots
- **Sports** - Running and workout gear
- **Winter** - Snow and cold weather equipment
- **Hygiene** - Personal care items
- **Medicine** - Health and first aid
- **Documents** - Important papers and IDs (highlighted in red)
- **Electronics** - Devices and chargers
- **Other** - Miscellaneous travel essentials

## Usage

### Online
- Estonian version: Open `travel-checklist.html` in your browser
- English version: Open `travel-checklist-en.html` in your browser

### Features
- **Check items** - Click any item to mark it as packed
- **Clear All** - Reset all checkmarks with one button
- **Sort** - Organize items by showing unpacked items first, then packed items

## Local Storage

The app uses browser's localStorage to save your progress. Each language version maintains its own separate checklist state, so you can use both independently.

## Technologies

- Pure HTML/CSS/JavaScript
- No dependencies or frameworks required
- Modern, gradient-based UI design
- LocalStorage API for persistence
